# Portfolio
A hub of links to some of my interactive works, easily expandable and (semi-)adaptive.

**This is the old version, which is frankly a bit of a nightmare all around. Uploaded for archiving purposes**


### List of works used:
* Click The Eyes
  - An exercise in interactive programming created using Processing.js
  - The eyes' pupils follow the cursor's movement using mouse tracking and exponent equasions
  - The eyes display a primitive squinting animation if clicked

* Twitter UX Review Poster:
  - An informational poster with a comprehensive review of Twitter from a UX standpoint
  - Made just before the Elon Musk takeover, is probaly a bit outdated

* Tripp3r navigation app:
  - A fully interactive Figma prototype of a navigation app for a Dutch railway company
  - Made with an Android phone in mind
  - Includes route descriptions and an interactive map

* AR Undyne Simulator:
  - A (very janky) Undertale fan-game made using P5.js and ML5.js
  - Tracks the player's body and hand movements through their camera using computer vision
  - Use the shield between your hands to block the incoming homing arrows or try to dodge them

* Bubble Clock:
  - An interactive P5.js clock with a primitive physics simulation and detailed collisions
  - Press and hold anywhere on the screen to blow a bubble
  - Larger bubbles rise slower and are harder to be pushed around by other bubbles
